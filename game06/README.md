##解題方式

先將 `WelcomeToTheJungle.wav` 用 Audacity 以頻譜方式打開，會發現是一個 dropbox 連結

![](screenshot.png)

下載 dropbox 連結後是另外一個音訊檔案 `FinalCountDown.mp3` ，然後音訊中有一個段落為摩斯電碼，將摩斯電碼解譯即為 Flag
